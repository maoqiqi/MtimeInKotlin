# MtimeInKotlin
用charles抓包时光网API资源，做成一个android客户端，包括正在热映，即将热映，搜索，榜单，预告片等。用kotlin+rxjava+retrofit+Material Design风格，主要练手kotlin语法。想看与已看使用本地数据库存储，ROOM库。无商业目的，纯练手。。。。
